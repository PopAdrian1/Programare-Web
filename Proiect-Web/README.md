# 🍎 Optimizarea Livezii — Algoritmi Genetici + Aplicație Web

## Descrierea Proiectului

Proiectul combină două componente principale:

### 1. Motor de Optimizare (Python)
Algoritm genetic care determină plasarea optimă a pomilor
într-o livadă pentru maximizarea producției, luând în considerare:
- Tipul solului
- Expunerea la soare
- Distanța dintre pomi
- Compatibilitatea dintre soiuri

### 2. Aplicație Web (Node.js + HTML/CSS/JS)
Interfață web care vizualizează rezultatele algoritmului genetic:
- Afișarea hărții livezii generate
- Statistici despre producția estimată
- Comparație între generații ale algoritmului
- Autentificare cu cont de utilizator
